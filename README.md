# YaVoice

Синтезатор речи с использованием сервисов Яндекс. Необходимо получить API-ключ для использования [здесь](https://developer.tech.yandex.ru/).


### Использование

```sh
yavoice -k <api-ключ> -t "Мой дядя самых честных правил" -f wav -v oksana -p
```
